### >>> Comandos Básicos <<<

Tipos numéricos:
  <p> **int**   inteiro
  <p> **float**    decimal

Operadores matemáticos:
* +  adição
* -  subtração
* +  multiplicação
* **  potência
* /  divisão ( sempre resultará em um número decimal)
* //  divisão com resultado inteiro
* %  "resto da divisão"</p>

````programming
>>> 2+3
5
>>> 6-2
4
>>> 7*3
21
>>> 2**5
32
>>> 4/2
2.0
>>> 5/2
2.5
>>> 2/3
0.6666666666666666
>>> 5/3
1.6666666666666667
>>> 7/3
2.3333333333333335
>>> 4//2
2
>>> 5//2
2
>>> 4%2
0
>>> 5%2
1
>>> 
````
